# A11yGate™ Modules
_Version: 0.1 · Source: Brand Book v3 §7_

A11yGate is modular accessibility proof infrastructure.
The master identity is "accessibility proof infrastructure for governed digital systems."
Modules are subcategories — they never supersede the master identity.

## Active Modules

### Module 1 — Verified Readability (Reference Preview v0.1)

**Canonical subcategory name:** Verified Readability Infrastructure™

Scope: Typography risk scoring, font disambiguation validation, render delta reporting.
Primary product: A11yGate Reference Font™ + A11yGate Font Audit™ + TRS.
Status: Reference preview — weights provisional (AG-DEC-0005).

## Identity Rule

No module name may appear in the site hero, meta description, or brand tagline.
The master identity always leads. Module names are used in product pages, docs,
and technical contexts only.
