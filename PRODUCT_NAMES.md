## Three-Layer Font Architecture

The Reference Font, brand typography, and UI render fonts are three different things:

```
Layer 1 ‚ÄĒ Brand typography:    Playfair Display + Source Serif 4 + IBM Plex Sans
Layer 2 ‚ÄĒ Reference instrument: A11yGate Reference Font (Atkinson fork, OFL-1.1)
Layer 3 ‚ÄĒ UI render font:      Source Serif 4 / IBM Plex Sans / IBM Plex Mono
```

The Reference Font is a measurement instrument. It is never used for brand headings.
