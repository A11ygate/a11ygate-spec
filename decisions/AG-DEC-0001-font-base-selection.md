# AG-DEC-0001: Base font — Atkinson Hyperlegible
Date: 2026-05-03 | Status: ACCEPTED | Impact: font-spec.md §1
Decision: Atkinson Hyperlegible v2 (OFL-1.1) as A11yGate Reference Font base.
Rationale: OFL, research-backed (Braille Institute 2019), a11y-first, minimal design risk.
Rejected: Noto Sans (not a11y-specialized), APHont (restrictive license), custom (wrong phase).
